# 🎓 TC Generator System

### Generate • Manage • Automate Transfer Certificates

TC Generator System is a full-stack web application designed to automate the process of generating Transfer Certificates (TC) for students. The system allows administrators to enter student details and generate a formatted certificate digitally.

---

## 🚀 Project Overview

Educational institutions often generate Transfer Certificates manually, which can be time-consuming and error-prone.

This project provides a digital solution where administrators can quickly generate Transfer Certificates using a web-based interface.

🎯 Goal:
To develop an efficient web system that automates certificate generation using frontend, backend, and database technologies.

---

## ✨ Key Features

- 🎓 Automatic Transfer Certificate generation
- 📄 Structured certificate format
- 🧑‍🎓 Student information management

## ⚙️ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/arpitat09/tc-generator.git

2️⃣ Navigate to the project directory

cd tc-generator

3️⃣ Install dependencies

npm install

4️⃣ Start the server

node server.js

5️⃣ Open the application in your browser

http://localhost:3000
